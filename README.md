# Rapid Recovery: Elite Training Center

Official website for Rapid Recovery: Elite Training Center - Private 18+ Only Gym in Allentown, PA.

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/) - Web framework
- Static site generation for GitHub Pages

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site automatically deploys to GitHub Pages when changes are pushed to the `main` branch.

Live site: https://taytayfos.github.io/rapid-recovery-website/
